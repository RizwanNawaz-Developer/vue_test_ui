# Vue UI/UX Test
Vue form

### Requirements

- Node.js v20.x or greater
- npm v9.x+

### Installation

1. Clone the Git Repository
```bash
git clone https://github.com/RizwanNawaz-Developer/vue_ui_test.git
```

2. Install Project Dependencies
```bash
npm install
```

3. Run the Development Server
```bash
npm run dev
```

4. Output
```bash
http://localhost:5173/
```
